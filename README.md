# life

Claude Code のスキルを活用した日々の情報収集・整理ツール。

## スキル一覧

| スキル | 説明 |
|--------|------|
| `neta-trend-daily` | はてブ・Hacker News・Reddit からトレンドネタを収集 |
| `ai-trend-daily` | AI 分野に特化したトレンド情報を収集 |
| `url-digest` | 複数 URL のコンテンツを要約 |

## クレジット

`neta-trend-daily` および `url-digest` スキルは [hand-dot](https://github.com/hand-dot) さんが YouTube で紹介・公開されていた [Gist](https://gist.github.com/hand-dot/bf6f928dce14095d5eef4f6aae63275e) をベースにしています。
